The Satellite and Security

Because of its working idea, the Satellite is able to monitor and save the words you use often and make mistakes in. The collected words are used to create personalized exercises and they can also be seen on the statistics section of the program.

This is usually not a problem for single user installations, since the data is stored locally to your hard drive. The Satellite does not collect numbers or special marks.

However, if you work in a business or government organization where data security is very important, there are some security issues that have to be taken into account.

High Security

The Satellite itself does not fulfill high security requirements. However, high security can be obtained with the appropriate Windows NT/2000/XP/2003 security settings:

Nobody else has access to your Windows account nor knows your password
Only you have read privileges to your personal 'documents and settings' folder (in Custom Install cases to TypingMaster installation folder)
You always lock your computer whenever you leave your workstation

 

Security Tips

Do Not Collect Words (High Security) Setting
By choosing High Security option during the first Satellite launch, the Satellite will not store any words, it just collects statistics about your difficult keys.

Confidential Applications
If you type confidential information in one or more applications, you can prevent the Satellite from monitoring them from the Satellite Settings dialog.

Confidential Documents
If you are typing a single document containing confidential information, you can temporarily disable the Satellite by clicking on the Satellite icon on the task bar and unchecking 'Enabled'.

Confidential Words
If any confidential words are visible in the list of difficult words in the statistics section, you can hide them permanently with the Hide Word command.

Uninstallation
The Satellite is removed when you uninstall TypingMaster Pro. You can also disabled it by clicking on the Satellite icon in the task bar.

 

Security Statement

The Satellite is able to monitor typing and reconstruct the typed words. The Satellite is not able to reconstruct sentences, it only saves single words.

The Satellite saves the collected data in the TypingMaster 'database' folder and uses the data in the program to show statistics and create personalized exercises. The Satellite does not use this data in any other way nor send it anywhere.

The Satellite is not able to monitor the Windows logon password nor network logon passwords.
